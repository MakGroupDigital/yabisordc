# **App Name**: Ya Biso RDC

## Core Features:

- Immersive Home: Display a background video and predictive search bar for popular destinations.
- Search and Filter: Enable users to search for places with real-time filters (price, city, type).
- Booking Tunnel: Guide users through the booking process, including date selection, options, payment, and QR code ticket generation.
- User Dashboard: Provide a dashboard for users to manage their trips, favorites, and tickets.
- Admin Panel: Secure interface for managing places (CRUD operations).

## Style Guidelines:

- Primary color: Congo Blue (#003366) for headers and main texts.
- Action/CTA: Ya Biso Orange (#FF8800) for call-to-action buttons.
- Nature: Virunga Green (#339966) for eco tags and success messages.
- Accent: Mineral Gold (#FFCC00) for stars and premium accents.
- Headline font: 'Montserrat' (sans-serif) for titles.
- Body font: 'Inter' (sans-serif) for text.
- Modern and airy designs with rounded corners (rounded-xl or 2xl) and soft shadows.
- Bento grid style for home page sections to display content in an organized and visually appealing manner.
- Use Framer Motion for smooth transitions and subtle animations (e.g., appearing elements on scroll, zoom on hover).
